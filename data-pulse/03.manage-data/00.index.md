# Managing Null Values & Data Cleaning

## Null Value Detection

### Identify Nulls
```python
df.isnull()                  # Returns True/False for each cell
df.notnull()                 # Opposite of isnull()
df.isnull().sum()            # Count nulls per column
df.isnull().sum(axis=1)      # Count nulls per row
df.isnull().any()            # Which columns have nulls
```

### Inspect Nulls
```python
df.info()                    # Shows non-null count & dtype
df.describe()               # Summary stats (excludes nulls by default)
```

---

## Handling Null Values

### Drop Nulls
```python
df.dropna()                  # Drop rows with any null
df.dropna(axis=1)            # Drop columns with any null
df.dropna(how='all')         # Drop row only if ALL values are null
df.dropna(thresh=2)          # Drop row if less than 2 non-null values
df.dropna(subset=['colA'])   # Drop rows where 'colA' is null
```

### Fill Nulls
```python
df.fillna(0)                 # Fill all nulls with 0
df.fillna(method='ffill')    # Forward fill (propagate previous value)
df.fillna(method='bfill')    # Backward fill (use next value)
df.fillna(df.mean())         # Fill with column mean
df.fillna(df['col'].median())# Fill with column median
df.fillna(df['col'].mode()[0]) # Fill with column mode
df.fillna({'A': 0, 'B': 'unknown'}) # Different fill per column
```

### Interpolate
```python
df.interpolate()            # Linear interpolation (default)
df.interpolate(method='linear', limit_direction='forward')
```

### Replace Specific Values
```python
df.replace('N/A', np.nan)    # Convert 'N/A' strings to null
df.replace([0, -999], np.nan) # Replace 0 and -999 with null
```

## Data Cleaning

### Duplicates
```python
df.duplicated()             # Returns True for duplicate rows
df.duplicated().sum()       # Count duplicates
df.drop_duplicates()        # Remove duplicate rows
df.drop_duplicates(subset=['colA', 'colB']) # By specific columns
```

### String Cleanup
```python
df['col'] = df['col'].str.strip()           # Remove whitespace
df['col'] = df['col'].str.lower()           # Lowercase
df['col'] = df['col'].str.upper()           # Uppercase
df['col'] = df['col'].str.replace('old', 'new') # Replace substring
```

### Type Conversion
```python
df['col'] = df['col'].astype(str)    # Convert to string
df['col'] = df['col'].astype(int)    # Convert to integer
df['col'] = pd.to_numeric(df['col'], errors='coerce') # Force numeric
df['date'] = pd.to_datetime(df['date']) # Convert to datetime
```

### Outliers
```python
# IQR method
Q1 = df['col'].quantile(0.25)
Q3 = df['col'].quantile(0.75)
IQR = Q3 - Q1
df[(df['col'] >= Q1 - 1.5*IQR) & (df['col'] <= Q3 + 1.5*IQR)]
```

### Reset Index
```python
df.reset_index(drop=True)   # Reset after dropping rows
```

## Quick Reference

| Operation | Method |
|-----------|--------|
| Find nulls | `isnull()`, `notnull()` |
| Count nulls | `isnull().sum()` |
| Drop rows | `dropna()` |
| Fill with value | `fillna(value)` |
| Fill forward | `fillna(method='ffill')` |
| Remove duplicates | `drop_duplicates()` |
| Replace values | `replace(old, new)` |
| Change dtype | `astype(type)` |
